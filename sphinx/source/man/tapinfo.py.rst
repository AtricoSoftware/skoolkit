==========
tapinfo.py
==========

SYNOPSIS
========
``tapinfo.py`` [options] FILE

DESCRIPTION
===========
``tapinfo.py`` shows information on the blocks in a TAP or TZX file.

OPTIONS
=======
-V, --version
  Show the SkoolKit version number and exit.

EXAMPLE
=======
Show information on the blocks in  ``game.tzx``:

|
|   ``tapinfo.py game.tzx``
